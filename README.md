# 📊 E-commerce Sales Dashboard (BigQuery + Looker Studio)

## 🚀 Project Overview

This project analyzes e-commerce sales data using **Google BigQuery** and visualizes insights using **Looker Studio**.

---

## 📌 Key Insights

* 💰 Total Revenue: 10.8M+
* 📦 Total Orders: 181K+
* 👥 Unique Customers: 79K+
